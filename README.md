# Titanic Data Analysis

## Project Overview

This project analyzes the Titanic passenger dataset to explore factors that influenced passenger survival.

## Objectives

* Clean and prepare the dataset using Python.
* Analyse passenger data using SQL.
* Create visualizations and summaries using Excel.
* Identify patterns related to survival rates.

## Tools Used

* Python (Pandas)
* SQL
* Microsoft Excel

## Project Workflow

1. Cleaned the raw dataset using Python.
2. Imported the cleaned data into SQL.
3. Performed exploratory data analysis using SQL queries.
4. Created charts and summaries in Excel.
5. Documented key findings.

## Key Questions

* Did gender affect survival rates?
* Did passenger class affect survival?
* Was age related to survival?
* Did ticket fare influence survival chances?

## Dataset

The dataset used in this project is the Titanic passenger dataset, which contains demographic and travel information for passengers aboard the RMS Titanic.
